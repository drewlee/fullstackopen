# Notes Frontend

This app is hosted on Vercel:
https://fso-notes-rouge.vercel.app/

## Development

Start the backend server:

```sh
$ cd ../../part3/notes-backend
$ npm run dev
```

Start Vite:

```sh
$ npm run dev
```

## Deployment

Deployment is handled via the
[notes-backend](../../part3/notes-backend/) app.
