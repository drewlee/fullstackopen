# Phonebook Frontend

This app is hosted on Vercel:
https://phonebook-backend-nu.vercel.app

## Development

Start the backend server:

```sh
$ cd ../phonebook-backend
$ npm run dev
```

Start Vite:

```sh
$ npm run dev
```

## Deployment

Deployment is handled via the
[phonebook-backend](../phonebook-backend/) app.
